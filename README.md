# Zrzr Portfolio

私のポートフォリオです。順次更新予定です。

Cloudflare Workersにデプロイしています。

## 技術スタック
- Vite + Reactをフルスタックフレームワークとして使用
- React Routerでルーティングを管理
- PandaCSS
- Biome.js
- TypeScript
